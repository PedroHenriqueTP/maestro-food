# 🚀 Projeto Maestro
*Serviço de Gestão Integrada para FoodService*

## 💡 A Missão
Centralizar a operação de FoodService no Brasil com orquestração inteligente e gestão de dados em tempo real.

## 🛠️ O Stack (Arquitetura de Alta Performance)
- **Framework:** Next.js 15 (Front) | NestJS (Back)
- **Database:** PostgreSQL (via Supabase) com RLS ativado.
- **Queue/Async:** BullMQ + Redis.
- **AI Integration:** Orquestração de agentes via Antigravity IDE.

## 🏃 Como começar
1. Clone o repo: `git clone ...`
2. Instale: `npm install`
3. Configure as envs (`.env.example` fornecido).
4. Rode a infra: `docker-compose up -d` (Redis/Postgres).

## 🧩 Protocolo de Contribuição
